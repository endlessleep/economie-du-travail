1.1 Intuition Fondamentale & Mathématiques

🧠 Intuition Derrière le Modèle

Avant de plonger dans des mathématiques complexes, il est essentiel de comprendre l'intuition et la logique du modèle. Le but est d'estimer et de simuler la trajectoire d'un actif financier sur le long terme tout en prenant en compte l'incertitude et la valeur temporelle de l'argent.

Définition : Actif Sous-Jacent

Un actif sous-jacent est un instrument financier (comme une action, une obligation ou une devise) sur lequel porte un produit dérivé (comme une option). Le prix final ou le gain du dérivé dérive entièrement de la valeur de ce sous-jacent.

⚙️ Modélisation Mathématique (Processus de diffusion)

Pour décrire comment le prix varie dans le temps, nous utilisons généralement une Équation Différentielle Stochastique (EDS) qui combine une tendance prévisible et un choc aléatoire.

Hypothèse de base : Mouvement Brownien Géométrique (GBM)

Nous faisons l'hypothèse très forte que le rendement de l'actif suit une distribution normale, ce qui implique que son prix suit une distribution log-normale. Cette subtilité est cruciale car elle garantit que le prix restera toujours strictement positif (une action ne peut pas valoir -10€).

\[ dS_t = \mu S_t dt + \sigma S_t dW_t \]
Où \( \mu \) est la dérive / taux de rendement espéré, \( \sigma \) est la volatilité, et \( dW_t \) est l'incrément brownien.

💻 Implémentation en Python (Simulation de Monte-Carlo)

En informatique, il n'y a pas de variables continues. Pour simuler ce modèle dans la "vraie vie", nous devons utiliser la solution exacte de l'EDS, discrétisée en petits sauts de temps \(\Delta t\). 

import numpy as np
import matplotlib.pyplot as plt

# 1. Paramètres initiaux
S0 = 100.0      # Prix initial de l'action ($)
mu = 0.05       # Taux de croissance attendu (5%)
sigma = 0.2     # Volatilité annuelle (20%)
T = 1.0         # Horizon de temps (1 an)
dt = 1/252      # Pas de temps (jours de trading)

# 2. Simulation des trajectoires
N = int(T / dt)
t = np.linspace(0, T, N)

# Génération d'un bruit blanc Gaussien (Chocs)
W = np.random.standard_normal(size=N) 
W = np.cumsum(W) * np.sqrt(dt)

# Solution exacte du Mouvement Brownien Géométrique
S = S0 * np.exp((mu - 0.5 * sigma**2) * t + sigma * W)

⚠️ Erreur Fréquente (Ajustement d'Itô)

Dans la formule exponentielle du code ci-dessus, il ne faut surtout pas oublier de soustraire \( \frac{1}{2}\sigma^2 \) à \( \mu \). C'est ce qu'on appelle "l'ajustement convexe d'Itô". Les étudiants oublient très souvent ce terme à l'examen !

🚀 Conseils pour briller à l'examen

💡 Exam Tip : Calcul des Probabilités

Si la question demande "Quelle est la probabilité que \( S_T > K \) ?", ne vous lancez pas dans des intégrales impossibles de loi log-normale ! Appliquez le log naturel de chaque côté pour retomber sur une loi normale centrée réduite classique : \( \mathcal{N}(0, 1) \).

📊 Pro Tip Excel / VBA

Pour calculer la valeur exacte de "N(d1)" ou "N(d2)" dans le tableur Excel (très utilisé en salle de marché), utilisez toujours la fonction intégrée =LOI.NORMALE.STANDARD.N(d1; VRAI).
